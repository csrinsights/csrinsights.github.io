CSRInsights Website - Setup Instructions
========================================

FILES INCLUDED:
---------------
1. index.html       - Homepage
2. services.html    - Services page
3. about.html       - About Us page
4. contact.html     - Contact page
5. styles.css       - All styling (CSS)
6. script.js        - All JavaScript functionality
7. README.txt       - This file

INSTALLATION:
-------------
1. Create a new folder on your computer (e.g., "CSRInsights-Website")
2. Copy ALL 6 files into this folder (index.html, services.html, about.html, contact.html, styles.css, script.js)
3. Make sure all files are in the SAME folder - do NOT create subfolders
4. Double-click index.html to open the website in your browser

FOLDER STRUCTURE:
-----------------
CSRInsights-Website/
    ├── index.html
    ├── services.html
    ├── about.html
    ├── contact.html
    ├── styles.css
    └── script.js
